# 学习路线图 / Learning Roadmap

## 阶段一：基础入门 / Phase 1: Getting Started (1-2 weeks)

- [ ] 理解 AI Agent 核心概念和基本架构
- [ ] 学习 LLM 基础：Prompt Engineering, Function Calling
- [ ] 完成 `01-fundamentals` 模块所有题目
- [ ] 动手体验 1-2 个 Agent 框架（推荐 LangChain + AutoGen）

## 阶段二：框架与模式 / Phase 2: Frameworks & Patterns (2-3 weeks)

- [ ] 深入学习主流框架的核心 API
- [ ] 掌握 ReAct, Plan-and-Execute 等经典模式
- [ ] 完成 `02-frameworks` 和 `03-design-patterns` 模块
- [ ] 用代码实现至少 3 种设计模式

## 阶段三：核心能力 / Phase 3: Core Capabilities (2-3 weeks)

- [ ] 深入 RAG 和记忆系统
- [ ] 掌握工具使用和 MCP
- [ ] 完成 `07-memory-and-rag` 和 `08-tool-use` 模块
- [ ] 构建一个带记忆和工具调用的完整 Agent

## 阶段四：系统设计 / Phase 4: System Design (2 weeks)

- [ ] 学习 Agent 系统架构设计
- [ ] 掌握多 Agent 协作模式
- [ ] 完成 `04-system-design` 和 `06-multi-agent` 模块
- [ ] 设计并实现一个多 Agent 系统原型

## 阶段五：生产就绪 / Phase 5: Production Ready (2 weeks)

- [ ] 学习 Agent 评估和安全防护
- [ ] 研究真实工业案例
- [ ] 完成 `09-evaluation-and-safety` 和 `10-real-world-cases` 模块
- [ ] 进行模拟面试练习
