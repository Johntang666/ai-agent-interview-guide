# 01 - 基础概念 / Fundamentals

## 题目列表 / Questions

### 概念理解 / Conceptual Understanding

1. **什么是 AI Agent？它与传统 Chatbot 有什么本质区别？**
   What is an AI Agent? How does it fundamentally differ from a traditional chatbot?

2. **描述 Agent 的核心循环：感知（Perception）→ 推理（Reasoning）→ 行动（Action）。**
   Describe the core Agent loop: Perception → Reasoning → Action.

3. **什么是 Agent 的"自主性"？如何衡量一个 Agent 的自主程度？**
   What is Agent "autonomy"? How do you measure the degree of autonomy?

4. **解释 LLM-based Agent 的基本架构组成。**
   Explain the basic architectural components of an LLM-based Agent.

5. **Agent 中的 "环境（Environment）" 指什么？举例说明不同类型的环境。**
   What does "Environment" mean in the Agent context? Give examples of different environment types.

### 核心组件 / Core Components

6. **Agent 的规划（Planning）能力包含哪些方面？**
   What aspects does an Agent's planning capability include?

7. **解释 Agent 中"工具使用"的概念及其重要性。**
   Explain the concept and importance of "tool use" in Agents.

8. **什么是 Agent 的记忆系统？短期记忆和长期记忆分别如何实现？**
   What is the Agent memory system? How are short-term and long-term memory implemented?

9. **如何理解 Agent 中的"反思（Reflection）"机制？**
   How do you understand the "Reflection" mechanism in Agents?

10. **对比 Agentic AI 与 Generative AI 的关键差异。**
    Compare the key differences between Agentic AI and Generative AI.

### 深入思考 / Deep Thinking

11. **为什么说 LLM 是 Agent 的"大脑"？LLM 的哪些能力使其适合作为 Agent 的核心？**
    Why is the LLM called the "brain" of an Agent? Which LLM capabilities make it suitable as the Agent core?

12. **Agent 的涌现行为（Emergent Behavior）是什么？如何看待这种现象？**
    What is emergent behavior in Agents? How should we view this phenomenon?

13. **当前 LLM-based Agent 的主要局限性有哪些？**
    What are the main limitations of current LLM-based Agents?
