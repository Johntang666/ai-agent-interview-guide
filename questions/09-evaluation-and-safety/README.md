# 09 - 评估与安全 / Evaluation & Safety

## 题目列表 / Questions

### Agent 评估 / Agent Evaluation

1. **如何评估一个 Agent 的整体性能？有哪些关键指标？**
   How do you evaluate an Agent's overall performance? What are the key metrics?

2. **对比 Agent 评测的主流 Benchmark：SWE-bench、WebArena、GAIA 等。**
   Compare mainstream Agent benchmarks: SWE-bench, WebArena, GAIA, etc.

3. **如何构建一个端到端的 Agent 评测管线？**
   How do you build an end-to-end Agent evaluation pipeline?

4. **"LLM-as-a-Judge" 评估方法的优缺点是什么？**
   What are the pros and cons of "LLM-as-a-Judge" evaluation?

5. **如何评估 Agent 的工具使用准确率和效率？**
   How do you evaluate an Agent's tool-use accuracy and efficiency?

### 安全防护 / Safety & Security

6. **什么是 Prompt Injection？Agent 系统中有哪些特有的注入风险？**
   What is Prompt Injection? What injection risks are unique to Agent systems?

7. **如何防止 Agent 执行危险操作（如删除文件、发送未授权请求）？**
   How do you prevent Agents from executing dangerous operations (deleting files, unauthorized requests)?

8. **Agent 的权限控制如何设计？最小权限原则如何落地？**
   How do you design Agent permission control? How do you implement least-privilege?

9. **如何实现 Agent 的输出安全过滤（Content Safety）？**
   How do you implement Agent output content safety filtering?

10. **对比不同的 Agent 安全架构：沙箱、审批流、guardrails。**
    Compare different Agent safety architectures: sandbox, approval flows, guardrails.

### 对齐与可信 / Alignment & Trust

13. **如何设计 Agent 的审计日志（Audit Log）系统？**
