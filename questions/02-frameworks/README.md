# 02 - 主流框架 / Frameworks

## 题目列表 / Questions

### 框架对比 / Framework Comparison

1. **对比 LangChain 和 LangGraph 的核心区别，各自适用什么场景？**
   Compare the core differences between LangChain and LangGraph. What scenarios is each suited for?

2. **AutoGen 的多 Agent 对话框架有什么设计特点？**
   What are the design characteristics of AutoGen's multi-agent conversation framework?

3. **CrewAI 的角色（Role）驱动设计有什么优势和局限？**
   What are the advantages and limitations of CrewAI's role-driven design?

4. **Dify 作为 low-code Agent 平台，与代码优先的框架相比有什么取舍？**
   As a low-code Agent platform, what trade-offs does Dify have compared to code-first frameworks?

5. **对比 OpenAI Assistants API、Claude Agent SDK 和 LangChain Agent 的设计哲学。**
   Compare the design philosophies of OpenAI Assistants API, Claude Agent SDK, and LangChain Agent.

### 框架深入 / Deep Dive

6. **LangGraph 中的状态图（StateGraph）是如何工作的？为什么选择图结构？**
   How does StateGraph work in LangGraph? Why choose a graph structure?

7. **如何在 LangChain 中实现自定义工具（Custom Tool）？**
   How do you implement a Custom Tool in LangChain?

8. **解释 AutoGen 中 AssistantAgent 和 UserProxyAgent 的协作模式。**
   Explain the collaboration pattern between AssistantAgent and UserProxyAgent in AutoGen.

9. **如何选择合适的 Agent 框架？你的技术选型标准是什么？**
   How do you choose the right Agent framework? What are your technical selection criteria?

10. **如果让你从零搭建一个 Agent 框架，你会如何设计核心抽象？**
    If you were to build an Agent framework from scratch, how would you design the core abstractions?

### 实践问题 / Practical

11. **在生产环境中使用 LangChain 遇到过哪些坑？如何解决？**
    What pitfalls have you encountered using LangChain in production? How did you solve them?

12. **如何实现 Agent 框架的可观测性（Observability）？**
    How do you implement observability in an Agent framework?
