# 06 - 多智能体协作 / Multi-Agent

## 题目列表 / Questions

### 通信与协调 / Communication & Coordination

1. **多 Agent 系统有哪些常见的通信模式？各自的优缺点是什么？**
   What are common communication patterns in multi-agent systems? What are their pros and cons?

2. **如何设计 Agent 之间的消息传递协议？**
   How do you design message-passing protocols between Agents?

3. **多 Agent 系统中如何处理冲突（Conflict Resolution）？**
   How do you handle conflict resolution in multi-agent systems?

4. **解释"共享黑板（Shared Blackboard）"模式在多 Agent 协作中的应用。**
   Explain the "Shared Blackboard" pattern in multi-agent collaboration.

### 拓扑结构 / Topologies

5. **对比中心化（Hub-and-Spoke）和去中心化（Peer-to-Peer）的多 Agent 拓扑。**
   Compare centralized (Hub-and-Spoke) and decentralized (Peer-to-Peer) multi-agent topologies.

6. **什么是层级化多 Agent 架构？如何设计任务分配策略？**
   What is hierarchical multi-agent architecture? How do you design task allocation?

7. **如何设计一个动态组队的多 Agent 系统？**
   How do you design a multi-agent system with dynamic team formation?

### 实战场景 / Practical Scenarios

8. **设计一个"软件开发团队"多 Agent 系统（PM、Architect、Developer、Tester）。**
   Design a "software dev team" multi-agent system (PM, Architect, Developer, Tester).

9. **设计一个多 Agent 辩论系统，通过正反方辩论得出更优答案。**
   Design a multi-agent debate system that reaches better answers through adversarial debate.

10. **如何防止多 Agent 系统陷入无限循环或死锁？**
    How do you prevent multi-agent systems from infinite loops or deadlocks?

11. **多 Agent 系统中的"社会模拟"（Social Simulation）有哪些应用？**
    What are the applications of "Social Simulation" in multi-agent systems?
