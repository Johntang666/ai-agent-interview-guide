# 03 - 设计模式 / Design Patterns

## 题目列表 / Questions

### 经典模式 / Classic Patterns

1. **详细解释 ReAct（Reasoning + Acting）模式的工作流程和优缺点。**
   Explain the ReAct (Reasoning + Acting) pattern workflow, pros and cons in detail.

2. **Plan-and-Execute 模式与 ReAct 模式有什么区别？各自适用什么场景？**
   What are the differences between Plan-and-Execute and ReAct? What scenarios suit each?

3. **解释 Reflexion 模式如何让 Agent 从错误中学习。**
   Explain how the Reflexion pattern enables Agents to learn from mistakes.

4. **LATS（Language Agent Tree Search）是如何结合树搜索与 LLM 的？**
   How does LATS (Language Agent Tree Search) combine tree search with LLMs?

5. **Chain-of-Thought（CoT）和 Tree-of-Thought（ToT）在 Agent 推理中分别扮演什么角色？**
   What roles do Chain-of-Thought (CoT) and Tree-of-Thought (ToT) play in Agent reasoning?

### 架构模式 / Architectural Patterns

6. **什么是 Router Agent 模式？如何设计一个高效的路由策略？**
   What is the Router Agent pattern? How do you design an efficient routing strategy?

7. **解释 Supervisor 模式在多 Agent 系统中的应用。**
   Explain the application of the Supervisor pattern in multi-agent systems.

8. **什么是 "Agent as a Tool" 模式？它如何实现 Agent 的层级化组织？**
   What is the "Agent as a Tool" pattern? How does it enable hierarchical Agent organization?

9. **对比串行（Sequential）、并行（Parallel）、和图（Graph）Agent 编排方式。**
   Compare Sequential, Parallel, and Graph Agent orchestration approaches.

10. **如何设计一个具备 self-correction 能力的 Agent？**
    How do you design an Agent with self-correction capability?

### 高级模式 / Advanced Patterns

11. **什么是 Agentic RAG？它与传统 RAG 有什么区别？**
    What is Agentic RAG? How does it differ from traditional RAG?

12. **解释 "Human-in-the-Loop" 模式在 Agent 系统中的设计要点。**
    Explain the key design points of "Human-in-the-Loop" in Agent systems.

13. **如何实现 Agent 的动态规划与重规划（Re-planning）？**
    How do you implement dynamic planning and re-planning in an Agent?

### 补充题目 / Additional Questions

14. **说下 ReAct 框架。它是如何将思维链和行动结合起来，以完成复杂任务的？你还了解 Plan-and-Solve 和 Reflection 吗？**
    Explain the ReAct framework. How does it combine reasoning and action to solve complex tasks? Are you also familiar with Plan-and-Solve and Reflection?

    In Agent design, planning is critical. What mainstream methods can give LLMs planning capability, such as CoT, ToT, and GoT?
