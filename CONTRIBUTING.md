# 贡献指南 / Contributing Guide

感谢你对本项目的关注！欢迎通过以下方式参与贡献。

Thank you for your interest! Contributions are welcome in the following ways.

## 贡献方式 / How to Contribute

### 添加新题目 / Add New Questions

1. 在对应的 `questions/XX-topic/README.md` 中添加题目
2. 题目需包含中英双语
3. 标注难度等级（如有）

### 提交参考答案 / Submit Solutions

1. 在 `solutions/` 目录下对应的文件中添加答案
2. 答案应有深度，不仅是"是什么"，更要有"为什么"和"怎么做"
3. 建议附带代码示例或架构图

### 补充学习资源 / Add Resources

1. 在 `resources/README.md` 中添加
2. 确保链接有效且资源质量高

## PR 规范 / PR Guidelines

- 分支命名：`feat/topic-description` 或 `fix/issue-description`
- Commit 信息清晰简洁
- 一个 PR 尽量聚焦一个主题

## 题目格式 / Question Format

```markdown
**中文题目描述**
English question description.
```

## 答案格式 / Answer Format

```markdown
### Q: 题目 / Question

**核心要点 / Key Points:**
- 要点 1
- 要点 2

**详细解答 / Detailed Answer:**
...

**代码示例 / Code Example:** (如适用 / if applicable)
...
```
